# Firebase Todo App demo

## Setup

- https://firebase.google.com/docs/web/setup?hl=ja
- https://firebase.google.com/docs/firestore/quickstart?hl=ja
- write firebase configration to [App.jsx:l8](src/App.jsx)


## Development

1. npm install
1. npm run dev

## Deploy

1. npm run build
1. firebase deploy
